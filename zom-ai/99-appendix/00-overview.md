# Appendix Zone

Operational guidance for developing, testing, and debugging the Zom AI system. This zone contains practical how-to documentation for contributors.

---

## Purpose

The Appendix zone serves developers who need to:
- Set up a local development environment
- Run and debug the AI system
- Test changes to the agent or frontend

For understanding **what** the system does, see [Foundation](../00-foundation/00-overview.md).
For understanding **how** it's built, see [Codebase](../10-codebase/00-overview.md).
