# Racho's New Games Pack (Canvas Prototypes)

This folder contains **5 standalone HTML prototypes** (no external libraries).  
Open any file in a browser (Chrome/Edge recommended):

- `gel_tower.html` — Jelly Jump–style vertical runner (combo + dash)
- `shard_rush.html` — Smash Hit–style corridor shatter sprint (ammo economy)
- `loop_dots.html` — Two Dots–style connect puzzle (loops clear all)
- `poly_pulse.html` — Polyforge–style timing + painting polygon faces
- `branch_flip.html` — ZigZag / The Branch–style tap-to-turn runner

## Notes
- Audio uses WebAudio and will only start after you click “Start”.
- Best scores are saved in `localStorage`.
